name: run
description: Start the Vite dev server so you can visually verify UI changes in the browser.
disable-model-invocation: false
---

Run `npm run dev` to start the Vite dev server. The app will be available at http://localhost:5173 (or the next available port if 5173 is taken).

After starting the server:
1. Note the URL printed in the terminal.
2. Tell the user the app is running and share the URL.
3. If asked to verify a visual change, describe what to look for at that URL.

To stop the server, the user can press Ctrl+C in the terminal where it's running.
